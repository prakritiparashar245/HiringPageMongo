This is a simple application for saving HTML form data to MongoDB through NodeJS.

The public folder contains the views in which index.html is the root file or the 1st file to opened.

The server.js file contains the code for saving data in MongoDB collection. The databse's name is Comunev and collection's name is hiring.

The port which application runs on is 3000. the default address would be http://localhost:3000/public/index.html

There is another file named getalldata.js for viewing data stored in database(just for utility).

The files or resume will be uploaded in folder named uploads.
